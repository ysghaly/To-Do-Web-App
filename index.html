To-Do Web Application using jQuery

By: Youssef Ghaly

Instructions:

  • Use form to create new tasks. Newly created tasks will automatically appear in list. Order number is automatically calculated and changes when moving tasks up and down.
  • Click Show/Hide button to toggle tasklist visibility. Note: If list is hidden, creating a new task will make it visible.
  • Click Clear Tasks button to delete all tasks.
  • To delete individual tasks, click the delete button to its right.
  • Reorder a task up and down the list by clicking Up and Down buttons or by dragging them.
  • To Edit individual tasks, double click on the task, which will switch the form to an edit form. Note: You no longer need to re-enter old values, fields left blank will remain with previous values.
  • Click "Begin" to start.
